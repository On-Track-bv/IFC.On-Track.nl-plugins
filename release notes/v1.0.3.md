# 🎨 IFC On-Track bSDD plugin v1.0.3 - bSDD UX Restore & HiDPI Icons

**Release Date:** June 24, 2026

Patch release restoring the classic bSDD Revit plugin experience for users migrating from the original bSDD Revit plugin, with improvements for high-DPI displays and build system housekeeping.

## ✨ What's New

### Familiar bSDD Interface (Migration from bSDD Revit plugin)
Users migrating from the original bSDD Revit plugin will now find a familiar experience:

- ✅ Dedicated **"bSDD"** tab in the Add-Ins ribbon (was "IFC.On-Track.nl")
- ✅ Original **bSDD icons** restored for the selection and IFC export buttons
- ✅ Button tooltips match the original plugin wording

### HiDPI / 4K Display Support
- ✅ 96×96 pixel icons added for crisp display on high-DPI and 4K monitors
- ✅ Revit scales down cleanly on standard displays — no visual regression

### Installer
- ✅ Installer product name updated to **"IFC On-Track bSDD plugin"** (visible in Apps & features)
- ✅ Installer now correctly builds as Release in Release solution configurations

## 🐛 What's Fixed

- Fixed nullable reference warnings in `BsddSearchBridge` and `BsddSelectionView` with proper null guards
- Fixed `.addin` manifest assembly path to use Windows backslash separator
- Aligned `build.sh` temp directory with `build.ps1` (`.build/temp`)

## 📦 Installation

### Fresh Install
1. Download `IfcOnTrack.Revit-v1.0.3-SingleUser.msi` (no admin required) or `MultiUser.msi` (system-wide, admin required)
2. Run the installer and select your Revit version(s) (2025 and/or 2026)
3. Launch Revit — look for the **"bSDD"** tab in the Add-Ins ribbon

### Upgrade from v1.0.2
Uninstall the previous version first, then install v1.0.3. Settings stored in Revit project files are preserved.

### Upgrade from original bSDD Revit plugin
Uninstall the old plugin, then install v1.0.3. The ribbon tab name and icons will be familiar.

```powershell
# Verify installation
Test-Path "$env:APPDATA\Autodesk\Revit\Addins\2025\IfcOnTrack.Revit.addin"
Test-Path "$env:APPDATA\Autodesk\Revit\Addins\2025\IfcOnTrack.Revit\IfcOnTrack.Revit.dll"
```

## 🔄 Upgrade Notes

- **From v1.0.0–v1.0.2:** Uninstall old version first, then install v1.0.3
- **Settings Preserved:** ExtensibleStorage data in Revit files remains intact
- **No Breaking Changes:** Fully compatible with projects created in previous versions

## 📋 What's Included

All features from v1.0.0 remain available:
- bSDD classification panel and search
- Revit 2025 & 2026 support
- IFC export with classifications
- WebView2-based UI
- Multi-document support
- Auto-update notifications
