# 🔧 IFC.On-Track.nl Plugins v1.0.1 - Installer Fix

**Release Date:** June 22, 2026

Critical patch release fixing installer issues that prevented the plugin from being detected by Revit.

## 🐛 What's Fixed

### Critical Installer Bug
The v1.0.0 installer had a critical bug where:
- ❌ The `.addin` manifest file was not included in the installation
- ❌ Plugin files were installed directly in `Addins\2026\` instead of `Addins\2026\IfcOnTrack.Revit\`
- ❌ Revit could not detect the plugin

**Result:** Plugin was installed but invisible to Revit - no "IFC.On-Track.nl" tab in Add-Ins ribbon.

### What's Fixed in v1.0.1
- ✅ `.addin` manifest file now correctly installed
- ✅ Files installed in proper subfolder structure (`Addins\2026\IfcOnTrack.Revit\`)
- ✅ Revit now correctly detects and loads the plugin

## 📦 What's Included

- **IfcOnTrack.Revit-v1.0.1-SingleUser.msi** - Installs to user profile (no admin rights needed)
- **IfcOnTrack.Revit-v1.0.1-MultiUser.msi** - Installs system-wide (requires admin)

Both installers support **Revit 2025 and 2026** with feature selection during installation.

## 📥 Installation

### For New Users
Follow the installation steps from the [v1.0.0 release notes](Revit_v1.0.0.md#-installation).

### For Users Who Installed v1.0.0

If you installed v1.0.0 and the plugin didn't appear in Revit:

1. **Uninstall v1.0.0** (optional but recommended)
   - Windows Settings → Apps → IFC.On-Track.nl → Uninstall

2. **Install v1.0.1**
   - Download and run the new installer
   - Select your Revit versions (2025/2026)

3. **Verify Installation**
   ```powershell
   # Check that .addin file exists
   dir "$env:APPDATA\Autodesk\Revit\Addins\2026\IfcOnTrack.Revit\*.addin"
   ```

4. **Start Revit** - You should now see "IFC.On-Track.nl" in the Add-Ins ribbon

## 🔍 Technical Details

### Before (v1.0.0)
```
%AppData%\Autodesk\Revit\Addins\2026\
├── IfcOnTrack.Revit.dll              ❌ Wrong location
├── IfcOnTrack.Revit.deps.json
├── publish\
├── runtimes\
└── UI\
```
**Missing:** `.addin` file, subfolder structure

### After (v1.0.1)
```
%AppData%\Autodesk\Revit\Addins\2026\
└── IfcOnTrack.Revit\                 ✅ Correct subfolder
    ├── IfcOnTrack.Revit.addin        ✅ Manifest present
    ├── IfcOnTrack.Revit.dll
    ├── IfcOnTrack.Revit.deps.json
    ├── IfcOnTrack.Revit.runtimeconfig.json
    ├── publish\
    ├── runtimes\
    └── UI\
```

### Changes Made

1. **Project file** (`IfcOnTrack.Revit.csproj`)
   - Added `.addin` file to build output via `CopyToOutputDirectory`

2. **Installer generator** (`Installer.Generator.cs`)
   - Wrap files in `IfcOnTrack.Revit` subfolder
   - Include subdirectories (`publish\`, `runtimes\`, `UI\`)
