# 🔧 IFC.On-Track.nl Plugins v1.0.2 - Build System Fix

**Release Date:** June 22, 2026

Patch release fixing CI/CD build system issues and ensuring correct .addin file placement.

## 🐛 What's Fixed

### Build System Issues
The v1.0.1 release had build system problems that prevented successful MSI creation in CI/CD environments:

- ❌ MSI installer build failed on paths with dots (e.g., `Release.R25`)
- ❌ WixSharp wildcard patterns caused errors in GitHub Actions
- ❌ Duplicate WiX identifiers for nested runtime directories
- ⚠️ .addin file placement issue still present from v1.0.1

### What's Fixed in v1.0.2
- ✅ MSI installer now builds successfully in all environments
- ✅ Replaced WixSharp wildcards with explicit file enumeration
- ✅ Fixed duplicate identifiers by using full relative paths
- ✅ `.addin` file now correctly placed at `Addins\2025\` level (not inside subfolder)
- ✅ CI/CD pipeline fully operational for automated releases

### Technical Details

**Problem:** WixSharp's `DirFiles` with wildcard patterns (`*\*.*`) failed on Windows paths containing dots:
```
Error: The filename, directory name, or volume label syntax is incorrect:
'source\dotnet\IfcOnTrack.Revit\bin\Release.R25\*'
```

**Solution:**
- Recursively enumerate all files and directories explicitly
- Generate unique WiX identifiers using full relative paths
- Replace invalid characters (`.`, `-`) in identifiers with underscores

## 📦 Installation Structure

The installer now creates the **correct** Revit add-in structure:

```
%AppData%\Autodesk\Revit\Addins\
└── 2025\
    ├── IfcOnTrack.Revit.addin          ← Manifest file (Revit reads this)
    └── IfcOnTrack.Revit\
        ├── IfcOnTrack.Revit.dll
        ├── IfcOnTrack.Revit.deps.json
        └── runtimes\
            └── ... (runtime dependencies)
```

## 📥 Installation

### For All Users

This release supersedes v1.0.0 and v1.0.1. Follow these steps:

1. **Uninstall Previous Version** (if installed)
   - Windows Settings → Apps → IFC.On-Track.nl → Uninstall

2. **Download Installer**
   - **SingleUser.msi** - Installs to user profile (recommended, no admin needed)
   - **MultiUser.msi** - Installs system-wide (requires admin rights)

3. **Run Installer**
   - Select Revit versions to install (2025 and/or 2026)
   - Complete installation wizard

4. **Verify**
   - Launch Revit
   - Check for "IFC.On-Track.nl" tab in Add-Ins ribbon
   - Or verify files:
     ```powershell
     # Check .addin file exists at correct location
     Test-Path "$env:APPDATA\Autodesk\Revit\Addins\2025\IfcOnTrack.Revit.addin"
     
     # Check plugin folder exists
     Test-Path "$env:APPDATA\Autodesk\Revit\Addins\2025\IfcOnTrack.Revit\IfcOnTrack.Revit.dll"
     ```
