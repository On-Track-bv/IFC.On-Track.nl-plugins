# IFC.On-Track.nl Plugin Icons

Place icon files here:

- `bsdd-16.png` - 16x16 bSDD icon
- `bsdd-32.png` - 32x32 bSDD icon
- `ids-16.png` - 16x16 IDS icon
- `ids-32.png` - 32x32 IDS icon
- `settings-16.png` - 16x16 Settings icon
- `settings-32.png` - 32x32 Settings icon

Icons should be PNG format with transparency.
